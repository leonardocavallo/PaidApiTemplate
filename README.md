## ✨ Features

- **Secure Discord OAuth Login**
- **Create & Delete API Keys** (salted & hashed with bcrypt)  
- **User Balance with Paid API Calls**
- **[BETA] Recharge Balance via Crypto Payments (Coinbase Commerce Gateway, optional)**
